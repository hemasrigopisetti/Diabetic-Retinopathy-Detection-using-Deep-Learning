# 🩺 Diabetic Retinopathy Detection using Deep Learning

![Python](https://img.shields.io/badge/Python-3.8-blue)
![Status](https://img.shields.io/badge/Project-Completed-brightgreen)

---

## 📌 Overview

This project uses deep learning techniques to detect diabetic retinopathy from retinal fundus images. It assists in early diagnosis and classification of disease severity.

---

## 🚀 Features

✔ Automated disease detection\n
✔ Deep learning-based classification
✔ User-friendly interface
✔ Accurate prediction system

---

## 🛠️ Tech Stack

* Python
* PyTorch / TensorFlow
* OpenCV
* Flask
* Jupyter Notebook

---

## 📂 Project Structure

```bash
backend/   → Model training & prediction  
frontend/  → UI implementation  
```

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
jupyter notebook
```

## 📊 Results

The model classifies retinal images into:

* Normal
* Mild
* Moderate
* Severe
